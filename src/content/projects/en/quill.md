## Overview

Quill is an AI-powered system-wide technical dictionary application available for both macOS and Windows. It provides instant, context-aware definitions and explanations for technical terms across any application.

## Technical Stack

- **macOS**: Native Swift application leveraging system APIs for global hotkey activation
- **Windows**: Built with Rust and Tauri for near-native performance with a web-based UI
- **AI Backend**: LLM integration for intelligent, context-aware definitions

## Key Features

- **System-wide activation**: Access definitions from any application via global hotkey
- **Context-aware**: Understands the context of selected text for more relevant definitions
- **Offline-capable**: Core dictionary available without internet connection
- **Cross-platform**: Consistent experience across macOS and Windows

## Open Source
