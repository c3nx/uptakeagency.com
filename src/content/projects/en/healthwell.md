## Overview

Healthwell is a progressive web application demonstrating AI-powered health assessment capabilities. Users interact with a conversational AI that provides personalized wellness recommendations based on symptoms and health goals.

## Key Features

- **Conversational AI**: Natural language health assessment
- **PWA**: Installable on any device, works offline
- **Personalized recommendations**: Tailored wellness advice
- **Privacy-first**: No data stored on external servers
